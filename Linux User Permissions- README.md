# Linux User and Permissions Management Lab
## Overview
In this lab I practiced Linux user and file permission management using Ubuntu. 

## Environment
- Ubuntu Linux
- Virtual machine environment

## Tasks Completed
- Created new users using the adduser command
- Created groups for user management
- Created directories using mkdir
- Changed file ownership using chown
- Modified file permissions using chmod
- Tested read, write, and execute permissions

## Commands Practiced
- mkdir
- chown
- chmod
- adduser
- groups
